POST and View URLs
The POST URL indicates the CGI data sink on your web site. The View URL (optional) indicates where others go to track your location. (tap-and-hold to change)

Continuous Updates
Subscribing to continuous GPS updates will be harder on the battery since the GPS unit will be awake all the time, but it will usually be more accurate too.
Update Interval
The interval between posting or receiving a GPS fix and asking for or posting the next one.
Buffer Size
The application stores readings locally until they're confirmed as accepted by the webserver.
Startup Tracking Enabled
If this option is set to on, tracking will start immediately when the app opens.
