JSON Location Tracker

Post URL

Until Google writes a Prē version of Latitude or documents the update proces, we (Prē owners) are kinda stuck to building our own Latitude-a-likes. Fortunately, Google Maps has an API that's pretty straight forward. There are samples in several languages on github (see the About card).

Anyway, the Post URL is is the full path of the CGI that should collect your JSON location posts. The format of the posts is listed in a file called "PROTOCOL" in the distribution (ibid).
